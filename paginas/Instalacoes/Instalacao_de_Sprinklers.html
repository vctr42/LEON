Qual a importância da instalação de um sistema de Sprinklers?

A instalação de sprinklers é um aspecto crucial para a segurança contra incêndios em edifícios residenciais, comerciais e públicos. Vamos explorar o que são os sprinklers, como funcionam e por que sua instalação é tão importante:

O que são Sprinklers?

Os sprinklers são pequenos chuveiros hidráulicos conectados a um sistema de bombeamento de água.

Em caso de incêndio, esses chuveiros são ativados automaticamente quando pequenos dispositivos de vidro (chamados de ampolas ou bulbos) se rompem devido ao aumento da temperatura ambiente.

O sistema de sprinklers inclui uma reserva técnica de água, sistema de bombeamento, válvula de governo e alarme, sistema de detecção de funcionamento, ponto de recalque, conexões, tubulações, drenos e os próprios sprinklers.

Quando Utilizar Sprinklers?

A necessidade de instalar sprinklers é determinada pelo Corpo de Bombeiros, conforme as normativas de cada estado.

Alguns estados, como Santa Catarina, exigem sprinklers para edifícios residenciais acima de 100 metros de altura descendente.

Em São Paulo e outros estados, não há exigência de sprinklers para edifícios residenciais

Como Funciona o Sistema de Sprinklers?

Os chuveiros automáticos (sprinklers) são montados com os seguintes elementos:

Corpo: Conecta-se à tubulação hidráulica e serve como base para os componentes.

Elemento Termo Sensível (Bulbo de Vidro): Quando atinge uma temperatura específica (geralmente 68°C), o líquido expansível dentro do bulbo rompe o vidro, liberando o fluxo de água.

Obturador: Serve como base para o bulbo e libera a passagem de água quando o sistema é acionado.

Elemento de Vedação: Evita vazamentos entre o corpo e o obturador.

Dreno no Sistema de Sprinklers:

O dreno é essencial para manutenção e testes de funcionamento.

Deve ser instalado em todos os andares e em todas as regiões divididas pelas válvulas de governo e alarme.

Lembre-se de que a instalação de sprinklers deve ser realizada por profissionais especializados para garantir a segurança em caso de incêndios
